## Contributing to SKY

Welcome to the **SKY** project! Whether you are a developer, a UI designer, or a privacy enthusiast, your help is what makes this project possible. SKY is built on the belief that AI should be private, local, and accessible to everyone without a subscription.

---

### How You Can Contribute

There are many ways to help improve SKY, ranging from writing code to improving documentation.

* **Model Optimization:** Help us test and refine prompts for the **TinyDolphin** model to improve its reasoning and memory.
* **UI/UX Design:** SKY thrives on its "Deep Sea" aesthetic. If you have ideas for cleaner animations or better CSS layouts, we’d love to see them.
* **Feature Requests:** Suggest new local integrations (e.g., local file reading, system stats, or dark mode toggles).
* **Bug Reports:** Found a glitch in the Django backend or a REST API timeout? Open an issue so we can squash it.
* **Documentation:** Help us make the setup guide even simpler for people who are new to **Ollama** or **Django**.

---

### Getting Started

1. **Fork the Repository:** Create your own copy of the project to work on.
2. **Clone Locally:** ```bash
git clone https://github.com/your-username/SKY.git
```

3. **Set Up Your Environment:** Ensure you have **Ollama** installed and the `tinydolphin` model pulled:
```bash
ollama pull tinydolphin

4. **Create a Branch:** Keep your changes organized.
```bash
git checkout -b feature/amazing-new-feature

```


5. **Submit a Pull Request:** Once you’re happy with your changes, send them over for review!

---

### Our Tech Stack

To contribute effectively, it helps to be familiar with:

* **Backend:** Python 3.14+ and Django.
* **Local LLM:** Ollama (specifically the TinyDolphin model).
* **Frontend:** HTML5, CSS3, and JavaScript (Vanilla or Framework-based).
* **API:** Django REST Framework for communication between the UI and the local model.
